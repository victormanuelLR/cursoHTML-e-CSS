Imagens dinâmicas 
use a tag picture e source para que as imagens do seu site se adaptem a cada usuário.
estrutura geral:
<picture>
	<source media='(max-width: ) srcset='' type='image/''>
	<source media='(max-width: ) srcset='' type='image/''>
	<img src='' alt=''>
</picture>

Áudios
<audio preload"matadata(recomendado)" controls>
	<source src='' type='audio/mpeg'>  
formatos que podem ser usados:
mp3, ogg, wav(não recomendado para rosteamento)
